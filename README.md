# Hotel Rajdhani

This is a beginner-friendly responsive hotel website developed as part of the **MANTRA 2026 Summer School Frontend Assignment**.

## Website Type

Hotel Website

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included

- Home
- About
- Rooms
- Facilities
- Gallery
- Booking
- Contact

## Features

- Responsive design for mobile, tablet, and desktop
- SEO meta tags on every page
- Image alt text for accessibility
- Responsive navigation menu
- Room booking form validation using JavaScript
- Room price calculator using JavaScript
- Gallery filter using JavaScript

## Hotel Facilities

- Luxury Rooms
- Multi-Cuisine Restaurant
- Swimming Pool
- Free Wi-Fi
- Free Parking
- 24/7 Room Service
- Conference Hall

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any web browser.

## Netlify Deployment Steps

1. Create a public GitHub repository.
2. Upload all project files to GitHub.
3. Log in to Netlify.
4. Click **Add new site** → **Import an existing project**.
5. Connect your GitHub repository.
6. Click **Deploy Site**.

## Project Description

**Hotel Rajdhani** is a responsive hotel website developed using **HTML5, CSS3, and JavaScript**. The website includes a Home page, About page, Rooms page, Facilities page, Gallery, Booking page, and Contact page. It features luxury room information, hotel facilities, a room price calculator, gallery filter, responsive navigation menu, and JavaScript form validation. This project was developed as part of the **MANTRA 2026 Summer School Frontend Assignment**.
